# Google AdSense 网站创建提示词

## 网站定位
创建一个专业的静态信息类英文网站，主题选择：科技资讯

## 技术要求
- 纯静态 HTML/CSS/JS 网站
- 响应式设计，适配移动端
- 页面加载速度优化
- 完整的 SEO 优化

## 必须包含的页面
1. **首页 (index.html)** - 展示最新文章列表
2. **文章详情页 (article.html)** - 单篇文章展示
3. **关于我们 (about.html)** - 网站介绍、团队信息
4. **联系我们 (contact.html)** - 联系方式、反馈表单
5. **隐私政策 (privacy-policy.html)** - 详细的隐私政策
6. **使用条款 (terms-of-service.html)** - 服务条款
7. **Cookie 政策 (cookie-policy.html)** - Cookie 使用说明
8. **免责声明 (disclaimer.html)** - 内容免责声明

## SEO 优化要求
- 每个页面设置唯一的 title 和 meta description
- 使用语义化 HTML5 标签（header, nav, main, article, footer 等）
- 添加 Open Graph 标签
- 添加结构化数据 (Schema.org)
- 图片添加 alt 属性
- 合理的 URL 结构
- 添加 sitemap.xml
- 添加 robots.txt

## 内容要求
- 至少 15-20 篇原创文章
- 每篇文章 800-1500 字
- 内容专业、有价值、无抄袭
- 文章配图清晰，标注来源
- 定期更新计划（每周至少2-3篇）

## 规范性页面内容要求

### 隐私政策 (Privacy Policy) 必须包含：
- 收集哪些用户信息
- 信息使用目的
- Cookie 使用说明
- 第三方服务（Google AdSense, Analytics）
- 用户权利（查看、修改、删除数据）
- 联系方式

### 使用条款 (Terms of Service) 必须包含：
- 网站使用规则
- 知识产权声明
- 责任限制
- 条款修改权利

### 免责声明 (Disclaimer) 必须包含：
- 内容准确性声明
- 外部链接声明
- 专业建议免责声明

## 设计规范
- 专业、简洁的设计风格
- 清晰的导航结构
- 合理的广告位预留（但申请前不要放置广告）
- 字体大小适中，阅读体验良好
- 配色协调，不超过3种主色

## AdSense 申请前检查清单
- [ ] 网站已上线并可访问
- [ ] 所有页面可以正常打开
- [ ] 没有占位符或空白页面
- [ ] 没有版权内容（图片、文字）
- [ ] 没有违规内容（成人、暴力、赌博等）
- [ ] 网站年龄超过1个月（建议）
- [ ] 有持续的原创内容更新
- [ ] 已添加 ads.txt 文件
- [ ] 已创建必要的规范页面

## 技术文件要求
1. ads.txt 文件（根目录）
2. robots.txt 文件
3. sitemap.xml 文件
4. 404.html 错误页面
