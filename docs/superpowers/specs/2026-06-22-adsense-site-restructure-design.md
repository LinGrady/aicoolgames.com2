# AdSense Site Restructure Design

Date: 2026-06-22

## Goal

Refresh Practical Tech Guide into a focused buying-advice publication that looks curated, current, and independently operated.

## Direction

Keep the static HTML structure and Practical Tech Guide identity. Rebuild public entry pages around real buying decisions rather than broad tech-news coverage. Add practical recent guides with grounded trade-offs and remove public emphasis from generic future-trend articles.

## Scope

- Do not modify `ads.txt`.
- Rebuild `index.html` and `categories.html` around buyer decision tracks.
- Refresh about, author, editorial, and contact pages.
- Add recent practical guide articles.
- Update `sitemap.xml` for public discovery.
- Leave old files in the repository, but stop presenting weak generic articles as primary content.

## Public Positioning

Practical Tech Guide helps readers compare tools, devices, and home-office technology. It should read like one careful editor maintaining a practical guide site, not a large fabricated newsroom or a generic AI news portal.
